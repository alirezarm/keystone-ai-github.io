Live Example

One question. Full answer.

See how Keystone turns a plain English question into a verified, multi-table SQL query with results — automatically.

keystone — agent chat
You
Which customer segments generate the most revenue over time?
Keystone Agent
I'll find the relevant tables and their join paths to answer this.
🧠 retrieve_schema 🔗 get_join_relationships ⚡ run_sql
SELECT c.segment,
  DATE_TRUNC('quarter', o.order_date) AS quarter,
  SUM(o.amount) AS total_revenue
FROM customers c
JOIN orders o ON c.customer_id = o.customer_id
GROUP BY c.segment, quarter
ORDER BY quarter, total_revenue DESC
